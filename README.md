## Installation
### Dependencies
* Gnome JavaScript (GJS)

### Run
1. Download
```
wget https://github.com/konkor/gjs-templates/archive/master.zip
```
2. Unpack this archive to any folder, _go to the example folder_
3. Run `./application`
